Interconnected Cost Function Networks (iCFN): an exact algorithm for multistate protein design with substate ensembles

Introduction

Interconnected Cost Function Networks (iCFN) is an exact algorithm for computational protein design in which you can solve different problems such as:

  • Side Chain packing

  • Protein design for single protein

  • Binding affinity toward multiple targets

  • Multistate protein design with single substate or multiple substates.

Pre-requisites and Installation

  • iCFN was developed and tested by C with gcc 4.4.7 compiler on CentOS 6.7 linux.

Usage

Multistate design example
./run_protein_design_TCR_Multistate.sh 26 2 2 5 10 > log_output &
Positive design example
./run_protein_design_TCR_positive.sh 26 2 2 5 10 > log_output &
Arguments
-ecutDEE             Real value       Specify energy cutoff for side chain packing and pruning by type dependant DEE
-ecutDEE_across      Real value       Specify energy cutoff for across substate type dependant DEE
-ecutDEE_seq         Real value       Specify specificity cutoff for ensemble fo sequences
-ecutDEE_stability   Real value       Specify stability cutoff for positive state from wild type
-max_dis_seq         Integer value    Specify maximum number of discrepancy sequence can have from wild type
-SNP_threshold       Integer value    Specify maximum SNP threshold sequence can have from wild type
-max_conf_seq        Integer value    Specify maximum number of conformation for each substate
-just_pos                             Specify condition to just solve positve protein design
